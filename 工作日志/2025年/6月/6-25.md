### 学到的东西

+ 泛型

+ 反射来使用私有方法：

```java
    private MMKV getMmkvByMmkvId(String mmkvId, boolean isDefault) throws DumpUsageException {
        if (!isDefault) {
            return MmkvUtils.getSingleMmkv(mmkvId);
        } else {
            // 默认使用反射调用私有的getDefaultMmkv()
            try {
                Class<?> mmkvUtilsClass = Class.forName("com.ktcp.video.util.MmkvUtils");
                Method getDefaultMmkvMethod = mmkvUtilsClass.getDeclaredMethod("getDefaultMmkv");
                getDefaultMmkvMethod.setAccessible(true);
                return (MMKV) getDefaultMmkvMethod.invoke(null);
            } catch (Exception e) {
                throw new DumpUsageException("获取默认MMKV实例失败: " + e.getMessage());
            }
        }
    }

```
