### 学习到的知识：

+ build.gradle.kts中的关键字：implementation、annotationProcessor、debugImplementation、releaseImplementation

+ debug、release模式以及其在build.gradle.kts中的配置以及AS中他们的构建选项

+ Application类：
    + MyApplication继承Application、在AndroidManifest.xml 中注册 Application 类
    + 全局变量存储：可以在 Application 类中定义全局变量，供全应用访问。
    + 第三方库初始化：如 Stetho、Retrofit、OkHttp、Firebase 等库的初始化都通常放在 Application 类中。
    + 管理资源：如数据库连接、日志工具等的初始化。

+ 高版本的Gradle需要明确配置才能生成 BuildConfig 文件
```java
// 启用 BuildConfig 生成
buildFeatures {
    buildConfig true
}
```
