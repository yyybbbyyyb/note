#### Http statusCode

304是没有修改过，可以读取缓存

基于 Java Android SDK 的 HttpURLConnection 类，这里是所有 HTTP 状态码的详细解析：

1xx 信息性状态码
（此类中未定义）

2xx 成功状态码
状态码	常量名	含义
200	HTTP_OK	请求成功，最常见的成功状态
201	HTTP_CREATED	资源创建成功（通常用于POST请求）
202	HTTP_ACCEPTED	请求已接受但未完成处理
203	HTTP_NOT_AUTHORITATIVE	非权威信息（代理返回的信息）
204	HTTP_NO_CONTENT	请求成功但无内容返回
205	HTTP_RESET	重置内容，要求客户端重置表单
206	HTTP_PARTIAL	部分内容（用于断点续传）
3xx 重定向状态码
状态码	常量名	含义
300	HTTP_MULT_CHOICE	多种选择，资源有多个可用表示
301	HTTP_MOVED_PERM	永久重定向
302	HTTP_MOVED_TEMP	临时重定向
303	HTTP_SEE_OTHER	查看其他位置
304	HTTP_NOT_MODIFIED	资源未修改（缓存有效）
305	HTTP_USE_PROXY	使用代理
4xx 客户端错误状态码
状态码	常量名	含义
400	HTTP_BAD_REQUEST	错误的请求语法
401	HTTP_UNAUTHORIZED	未授权，需要身份验证
402	HTTP_PAYMENT_REQUIRED	需要付费
403	HTTP_FORBIDDEN	禁止访问
404	HTTP_NOT_FOUND	资源未找到
405	HTTP_BAD_METHOD	方法不被允许
406	HTTP_NOT_ACCEPTABLE	不可接受的内容格式
407	HTTP_PROXY_AUTH	需要代理身份验证
408	HTTP_CLIENT_TIMEOUT	请求超时
409	HTTP_CONFLICT	冲突
410	HTTP_GONE	资源已不存在
411	HTTP_LENGTH_REQUIRED	需要内容长度
412	HTTP_PRECON_FAILED	前置条件失败
413	HTTP_ENTITY_TOO_LARGE	请求实体过大
414	HTTP_REQ_TOO_LONG	请求URI过长
415	HTTP_UNSUPPORTED_TYPE	不支持的媒体类型
5xx 服务器错误状态码
状态码	常量名	含义
500	HTTP_INTERNAL_ERROR	内部服务器错误
500	HTTP_SERVER_ERROR	⚠️ 已废弃，同上
501	HTTP_NOT_IMPLEMENTED	功能未实现
502	HTTP_BAD_GATEWAY	错误的网关
503	HTTP_UNAVAILABLE	服务不可用
504	HTTP_GATEWAY_TIMEOUT	网关超时
505	HTTP_VERSION	HTTP版本不支持


#### CGI

CGI 在网络请求中是 Common Gateway Interface（通用网关接口） 的缩写。

CGI 的含义
CGI 是一种标准协议，用于 Web 服务器与外部程序之间的通信：

作用: 允许 Web 服务器执行外部程序来处理 HTTP 请求
工作方式: 服务器接收请求 → 启动 CGI 程序 → 程序处理并返回结果 → 服务器返回给客户端
常见用途: 处理表单数据、数据库查询、动态内容生成

传统的 CGI 技术确实已经不常用了，因为：

性能问题: 每次请求都要启动新进程
资源消耗: 内存和 CPU 开销大
现代替代: FastCGI、WSGI、现代 Web 框架等
