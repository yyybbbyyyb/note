### 学到的东西

api gradle关键字：

+ implementation: 仅对当前模块可见，不会传递给其他模块。
+ api: 暴露给所有依赖当前模块的其他模块。
+ compileOnly: 编译时需要，运行时不需要。
+ runtimeOnly: 运行时需要，编译时不需要
+ testImplementation: 用于测试依赖。
+ androidTestImplementation: 用于Android测试依赖。
+ annotationProcessor: 用于注解处理器的依赖。

> 部分前面可以有debug和release的选择

allowBackup 属性的作用：
allowBackup 是 Android 应用的一个重要安全属性
它决定是否允许应用数据通过 ADB 备份/恢复
通常建议设置为 false 以防止敏感数据泄露
