#### 要将小的更改并入最近的 commit 且不影响提交顺序

+ git add .
+ git commit --amend --no-edit
    + --amend：将修改追加到最近 commit
    + --no-edit：保持原提交信息不变
+ git push origin master_@yaoboyu_story_125582732 --force-with-lease
    + --force-with-lease 比 -f 更安全，会在覆盖前检查远程是否有其

git show --name-only --oneline HEAD
